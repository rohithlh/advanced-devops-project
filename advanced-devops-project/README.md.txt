# Advanced DevOps Project

## Stack
- Terraform (Infra)
- Ansible (Config)
- Jenkins (CI/CD)
- Docker (Container)
- Kubernetes (Orchestration)
- AWS ECR (Registry)

## Features
- Remote backend (S3 + DynamoDB)
- IAM roles (secure)
- Blue-Green deployment
- ALB / Ingress
- Monitoring ready

## How to Run
1. Terraform apply
2. Ansible playbook
3. Jenkins pipeline
4. Access app via LoadBalancer
